Detailed Documentation

Enumerated potential types for APR process locking methods Check APR_HAS_foo_SERIALIZE defines to see if the platform supports APR_LOCK_foo. Only APR_LOCK_DEFAULT is portable.

Enum Values

APR_LOCK_FCNTL

fcntl()

APR_LOCK_FLOCK

flock()

APR_LOCK_SYSVSEM

System V Semaphores

APR_LOCK_PROC_PTHREAD

POSIX pthread process-based locking

APR_LOCK_POSIXSEM

POSIX semaphore process-based locking

APR_LOCK_DEFAULT

Use the default process lock
