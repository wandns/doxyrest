Detailed Documentation

basic memory node structure The next, ref and first_avail fields are available for use by the caller of apr_allocator_alloc() , the remaining fields are read-only. The next field has to be used with caution and sensibly set when the memnode is passed back to apr_allocator_free() . See apr_allocator_free() for details. The ref and first_avail fields will be properly restored by apr_allocator_free() .
